Tentang Aplikasi

Pokemon REST Client adalah aplikasi edukatif yang menampilkan data Pokémon menggunakan PokeAPI (https://pokeapi.co).

Fitur:

  • Menampilkan daftar Pokémon (20 item per halaman)
  • Pencarian berdasarkan nama
  • Detail lengkap tiap Pokémon (stats, abilities, sprites)
  • Sistem favorit sederhana menggunakan localStorage

Catatan teknis: Jika data tidak muncul saat membuka file langsung (file://), jalankan server lokal (mis. Live Server di VSCode) atau hosting sederhana sehingga fetch ke PokeAPI dapat berjalan.
